Note: To use this Application you will need a discord account

Additionally, you will need to go to the discord Developer portal (https://discord.com/developers/docs/intro) , You will then need set up a bot at the portal it will generate you a bot key.

At the bottom of this code you will need to enter that key create a discord test server and type node . in your IDE to activate the bot.

You can additionally, place it into a file that uses the key (better convention to follow).
